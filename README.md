Leap Device Driver
==================

Steps to Build and Run:
-----------------------
  1. Run "sudo leapd &" in the terminal
  2. Run "make clean"
  3. Run "make"
  4. Run "make install"
  5. Run "make run"
  6. Run "make uninstall"

Steps to Utilise:
-------------------
  1. insert to usb port
  2. place leap sensor upright on stable surface
  3. call leap in terminal
  4. utilise the UI to control pc volume, brightness etc
  5. exit from UI and unplug


Commands:
-----------
SWIPE LEFT:  Increase Volume
SWIPE RIGHT: Decrease Volume
SWIPE UP: Mute/UnMute
SWIPE DOWN:  Shows Available Disk Space
PINCH: Shows Available Memory 
GRAB: Display Uptime Of Your Device
OPEN: Displays Username Of Your Device
CIRCLE: Display the State, Config and Details of available network interfaces




Notes
-------
Ensure you have the "leapC" libary downloaded
https://www.ultraleap.com/downloads/leap-controller/
Ultraleap Hyperion in particular
